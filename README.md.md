### **Sales ETL Analysis Dashboard**

#### Project Overview

#### 

This project demonstrates an ETL-based Sales Data Analysis workflow using Python, SQL, and Excel to extract, transform, and visualize retail sales data. The goal of the project is to generate actionable business insights through data cleaning, query-based analysis, and an interactive Excel dashboard.

##### 

##### Tools \& Technologies



* Python (Data Extraction \& Preprocessing)

* SQL (Data Analysis Queries)

* Excel (Dashboard \& Visualization)

* Git \& GitHub (Version Control)

##### ETL Workflow

###### 1\. Extract

Loaded raw retail sales dataset using Python and SQL.

###### 

###### 2\. Transform

* Cleaned missing values and structured the dataset.

* Created derived fields such as Year and Month.

###### *Aggregated data to calculate:*

* Revenue by product

* Region-wise sales

* Month-over-month growth

* Profit growth by sub-category

###### 3\. Load

* Exported the transformed dataset into Excel.

* Built pivot tables and dashboards for visualization.

* Dashboard Insights

*The Excel dashboard answers key business questions:*

* Top 10 revenue-generating products

* Top products by region

* Month-over-month sales comparison (2022 vs 2023)

* Best-performing month for each category

* Sub-categories with highest and lowest profit growth

##### Key Findings

* Furnishings recorded the highest year-over-year profit growth.

* Paper showed the largest decline in profit.

* Sales performance varied significantly across regions and categories.
